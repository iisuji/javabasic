
  * 콜렉션 (표준화) : 함수가 고정화 됐구나... 
	- 기본적인 자료구조(개념)를 만들어 줌
	- java.util 패키지 
	
	1. List ==> 배열과 유사 (인터페이스) 
		- ArrayList : Object[] 동적크기 (배열 크기를 늘려줌)
		- 순서저장 (index 보유 ***) 
	2. Set
		- 순서 없음 ( index 없음 )
		- 중복값 저장 안 됨. (로또 같은 거 만들 때 중복된 거 배제하고 싶을 때 사용하면 유용)  
	3. Map 
		- 키(key)와 밸류(value) 저장 구조
		- 키는 중복 안됨
